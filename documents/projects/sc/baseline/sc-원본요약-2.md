# SC 디자인팀·영상팀 업무 도구 요구사항

## 요구 개요

SC 디자인팀·영상팀(전지연 이사)이 제출한 내부 업무 관리 도구 개선 요구다. 회의 내용을 자동으로 녹음·요약하고 관련자와 선택 공유하며, 구성원의 자리 비움 상태를 실시간으로 확인하고, 수정·신규 요청을 기존 작업과 연결하며, 업무의 진행·분배 상황을 확인하고 완료일 변경·임박을 작업자·요청자에게 동시 알람하는 것을 원한다. 전반적으로 요청자와 작업자 간 커뮤니케이션 효율을 높이고 업무 누락을 방지하려는 목적이다.

## 기능 목록

- [[sc-회의록-업무지시]] — 회의를 자동 녹음·요약하고 이전 회의록 참조·관련자 태그 선택 공유·도출 업무 진행 체크까지 연결하는 기능
- [[member-presence-status]] — 반차·연차·휴가·미팅·잠시 자리 비움 등 구성원의 현재 자리 상태를 실시간 확인하는 기능
- [[related-work-linking]] — 수정·신규 요청 시 이전/유관 작업을 연결하고 요청자·작업자·작업 경로를 함께 확인하는 기능
- [[sc-업무분장]] — 기획안 요청 후 담당 디자이너와 작업의 진행/예정 상태를 확인하는 업무 진행·분배 확인 기능
- [[sc-알림센터]] — 완료일 변경 및 완료일 D-1 임박 시 작업자·요청자에게 동시 알람을 발송하는 기능

## 원본 맥락

SC 디자인팀과 영상팀(전지연 이사)이 작성한 내부 업무 관리 도구 기능 요구사항 목록이다. 회의록 자동화, 자리 비움 확인, 연관 업무 연결, 업무 진행·분배 확인 및 알람의 4개 영역으로 구성되며, 각 영역은 팀 협업 과정에서 겪는 커뮤니케이션 비효율과 업무 누락 문제를 해결하려는 실무 요청이다.
