# 더에쓰씨 사내 통합 업무관리 시스템 개발 요청

## 요구 개요

더에쓰씨(THE SC) 경영관리부는 경영관리부·관련 TF·자회사의 업무를 하나로 묶는 사내 통합 업무관리 웹서비스 개발을 요청한다. 단순 게시판·일정관리가 아니라 업무관리·인사·근태·급여정산·계약·회계·컨택 등 실무 전 영역을 단일 시스템으로 통합하는 것이 목적이며, PC 웹을 우선하되 모바일 조회·등록·수정이 가능한 반응형 웹을 요구한다. 요청서는 현행 운영 시스템(팀앱)의 실제 규칙을 ▣ 블록으로 명시해 신규 개발 시 동일 결과를 요구하고, 야근수당·식대 계산 등 급여·정산 직결 로직은 ⚠️ 표시로 임의 변경을 금지하며 검증 기준값까지 제시한다. 전체 기능은 1~3차 단계별 개발로 견적을 구분하고, 소스코드·DB·디자인 소유권은 당사 귀속을 전제로 한다.

## 기능 목록

- [[sc-로그인-권한관리]] — 직원 계정 로그인과 조직·등급·민감정보 별도 권한 관리
- [[sc-메인-대시보드]] — 로그인 첫 화면의 업무 현황 집계와 통계 클릭·미니 캘린더 상호작용
- [[sc-회의록-업무지시]] — 회의록 기록과 결정사항 기반 업무 즉시 생성·수정 이력
- [[sc-서식함-자료실]] — 회사 양식·파일 업로드·버전관리·권한별 다운로드
- [[sc-직원명부-인사정보]] — 기본·근무·급여·외국인 정보와 직원 문서, 퇴사자 처리 규칙
- [[sc-근태-야근수당계산]] — 에스원 출입기록 업로드 기반 야근시간·수당 자동 계산(급여 직결 확정 규칙)
- [[sc-점심식대계산]] — 급식 CSV 업로드 기반 회사 분류·기간별 단가 식대 정산(정산 직결 확정 규칙)
- [[sc-계약서-법무관리]] — 표준계약서 빈칸 입력 생성과 계약 관리대장·만료 알림
- [[sc-회계자료관리]] — 회계 시트·파일·업무절차와 코드 기반 업무 흐름도 관리
- [[sc-업무분장]] — 직원별 담당업무 정리표 조회·엑셀 출력
- [[sc-인수인계]] — 퇴사·담당 변경 시 인수인계서 작성과 비밀번호 보안 전달
- [[sc-에이전시-컨택관리]] — 외부 파트너 컨택 현황 CRM과 상태·전환율 관리
- [[sc-레퍼런스보관함]] — 이미지·영상·URL 레퍼런스 저장과 태그·검색
- [[sc-추천답변-qna]] — 반복 응대 답변의 언어별 통합 저장·검색
- [[sc-통합검색]] — 시스템 상단 통합검색과 권한 기반 결과 필터
- [[sc-알림센터]] — 개인 알림센터와 능동적 리마인더·브로드캐스트 팝업
- [[sc-통계-리포트]] — 업무·인사·계약·비용·컨택 통계와 엑셀·PDF 출력
- [[sc-파일-데이터관리]] — 업로드 제한·미리보기·휴지통 복구·정기 자동 백업
- [[sc-보안-개인정보보호]] — 암호화·마스킹·접근권한·활동 로그 등 보안 정책
- [[sc-관리자설정]] — 조직·공통코드·상태값을 관리자가 직접 설정하는 기능

## 원본 맥락

더에쓰씨(THE SC) 경영관리부가 2026년 7월 작성한 "사내 경영관리 통합 업무관리 시스템 개발 요청서(통합 완성본)"이 출처다. 회계·인사·총무·법무·비서 파트로 구성된 경영관리부와 라이브·환자유치 에이전시 등 TF, 그리고 더데이랩스·메디솔브AI 등 자회사의 업무를 단일 시스템으로 통합하려는 목적에서 작성됐다. 요청서 본문에 현행 운영 시스템(팀앱)의 실제 구현 규칙을 각 장에 통합했고, 급여·정산 직결 로직에는 검증 기준값을 제시해 신규 개발이 동일 결과를 내도록 요구한다.
