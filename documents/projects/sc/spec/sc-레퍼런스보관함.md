# 레퍼런스보관함

> 한 줄 정의: 마케팅·기획 담당자가 SNS 콘텐츠·경쟁사 사례·광고 소재 등 이미지·영상·URL 레퍼런스를 등록하면, 제목·매체·카테고리·태그로 정리된 검색 가능한 보관함을 얻는다.

## 1. 요구 배경
- **현재 불편(as-is)**: 참고할 만한 SNS 콘텐츠, 경쟁사 사례, 광고 소재가 개인 기기나 여러 채널에 흩어져 있어 필요할 때 다시 찾기 어렵다. 요청서는 "SNS 콘텐츠, 경쟁사 사례, 광고 소재, 영상 및 이미지 레퍼런스를 저장하는 기능이 필요합니다"라고 명시한다.
- **필요 이유**: 레퍼런스를 한 곳에 등록·분류하고 검색·태그 필터·즐겨찾기로 다시 꺼내 쓸 수 있게 하여 자료 유실과 중복 탐색을 없앤다.

## 2. 기능 정의
- **등록한다**: 이미지 업로드, 영상 업로드, URL 등록, 화면 캡처 등록 방식으로 레퍼런스를 저장한다.
- **관리한다**: 제목·이미지/영상·원본 링크·매체·카테고리·태그·등록자·등록일·설명·활용 여부·관련 프로젝트를 항목으로 보관한다.
- **검색한다**: 저장된 레퍼런스를 검색어로 조회하고 태그 필터로 좁힌다.
- **즐겨찾기한다**: 자주 쓰는 레퍼런스를 즐겨찾기로 표시한다.
- **다운로드한다**: 저장된 이미지·영상 파일을 다운로드한다.

## 3. 유저 플로우 (반드시 표, 케이스별)

| 케이스 | 행동 (사용자가 한다) | 과정 (시스템이 처리) | 결과 (산출물) |
|---|---|---|---|
| 정상 | 담당자가 경쟁사 광고 이미지를 업로드하고 제목·매체·카테고리·태그·설명·관련 프로젝트를 입력한다 | 입력 항목과 파일을 저장하고 등록자·등록일을 기록한다 | 검색·태그 필터로 찾을 수 있는 레퍼런스 항목 1건 |
| 정상 | 담당자가 SNS 게시물 URL을 등록한다 | 원본 링크와 매체·카테고리·태그를 저장한다 | 원본 링크로 바로 이동 가능한 URL 레퍼런스 |
| 예외 | 담당자가 태그 필터로 특정 태그를 선택했으나 해당 태그의 레퍼런스가 없다 | 필터 조건에 맞는 항목을 조회한다 | 결과 없음 안내(빈 목록) |

## 4. 예시 (최소 2개, 실제 문장 인용)
- **입력**: "SNS 콘텐츠, 경쟁사 사례, 광고 소재, 영상 및 이미지 레퍼런스를 저장하는 기능이 필요합니다." — 담당자가 참고 광고 영상을 업로드하고 매체·카테고리·태그를 지정한다.
- **기대 출력**: 제목·매체·카테고리·태그·등록자·등록일이 붙은 영상 레퍼런스가 보관함에 저장되고, 태그 필터·검색·즐겨찾기·다운로드로 다시 꺼내 쓸 수 있다.
- **입력**: "화면 캡처 등록", "URL 등록" 방식으로 경쟁사 랜딩 페이지를 저장한다.
- **기대 출력**: 원본 링크와 캡처 이미지가 함께 저장되어 원본 링크로 이동하거나 캡처를 미리 볼 수 있다.

## 5. 상세 요구 (번호 목록, "~해야 한다")
1. 이미지 업로드, 영상 업로드, URL 등록, 화면 캡처 등록 방식으로 레퍼런스를 등록할 수 있어야 한다.
2. 모바일 공유 기능으로 레퍼런스를 등록할 수 있어야 하며, 브라우저 확장 프로그램 연동 가능 여부를 검토해야 한다.
3. 각 레퍼런스는 제목·이미지 또는 영상·원본 링크·매체·카테고리·태그·등록자·등록일·설명·활용 여부·관련 프로젝트를 관리 항목으로 가져야 한다.
4. 검색, 태그 필터, 즐겨찾기, 다운로드를 제공해야 한다.

## 6. 다루는 데이터 (해당 시)
- 입력/저장: 이미지·영상 파일, 원본 URL, 화면 캡처 이미지, 제목·매체·카테고리·태그·설명·활용 여부·관련 프로젝트·등록자·등록일.
- 출력: 검색·태그 필터 결과 목록, 즐겨찾기 목록, 다운로드 파일.

## 7. 수용 기준 (체크박스 최소 3개, 참/거짓 판정 가능)
- [ ] 이미지 업로드·영상 업로드·URL 등록·화면 캡처 등록 4가지 방식으로 각각 레퍼런스를 등록할 수 있다.
- [ ] 등록한 레퍼런스에 제목·매체·카테고리·태그·설명·활용 여부·관련 프로젝트가 저장되고 등록자·등록일이 자동 기록된다.
- [ ] 검색어로 레퍼런스를 조회하고 태그로 결과를 필터링할 수 있다.
- [ ] 레퍼런스를 즐겨찾기로 표시하고 저장된 파일을 다운로드할 수 있다.
