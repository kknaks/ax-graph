# 메인 대시보드

> 한 줄 정의: 직원이 로그인하면 / 시스템이 권한 범위에 맞춘 전체 업무 현황을 한 화면에 모아 보여줘 / 오늘 처리할 일과 임박·지연·만료 항목을 즉시 파악한다.

## 1. 요구 배경
- **현재 불편(as-is)**: 오늘·이번 주 일정, 마감 임박·지연 업무, 파트별 진행률, 계약·비자·라이선스 만료 예정 등이 각 메뉴에 흩어져 있어 현황을 한눈에 볼 수 없다. 요청서는 "로그인 후 첫 화면에서 전체 업무 현황을 확인할 수 있어야 합니다"라고 요구한다.
- **필요 이유**: 로그인 직후 첫 화면에 권한 범위 안의 핵심 현황을 모아 표시해 여러 메뉴를 돌지 않고도 즉시 처리 대상을 파악하게 한다.

## 2. 기능 정의
- **집계한다**: 오늘 일정·이번 주 일정, 마감 임박 업무, 마감 지연 업무, 담당자별 진행 업무, 파트별 업무 진행률을 로그인 첫 화면에 모아 표시한다.
- **경보한다**: 계약·비자 만료 예정, 소프트웨어 라이선스 만료 예정, 휴가·부재자 현황, 미확인 업무지시, 주요 알림을 표시한다.
- **권한 필터링한다**: 관리자는 전체 조직을, 일반사용자는 본인이 권한을 가진 파트와 본인 업무만 보이도록 표시 범위를 제한한다.
- **파트별 오늘 업무 통계를 낸다**: 파트별로 전체 / 시작 전 / 진행 중 / 완료 숫자를 표시하고(전체·완료는 오늘 기준), 통계 숫자 클릭 시 해당 상태의 업무 목록 팝업을 띄운다.
- **미니 캘린더를 제공한다**: 파트별 미니 캘린더에서 날짜 마우스 오버 시 그날 일정 툴팁을, 클릭 시 상세 팝업을 띄우고 '오늘 업무 현황' 섹션 접기/펼치기를 지원한다.

## 3. 유저 플로우 (반드시 표, 케이스별)

| 케이스 | 행동 (사용자가 한다) | 과정 (시스템이 처리) | 결과 (산출물) |
|---|---|---|---|
| 정상 | 일반사용자가 로그인한다 | 사용자 권한 범위(본인 파트·본인 업무)로 오늘·주간 일정, 마감 임박/지연, 파트별 진행률, 만료 예정 항목을 집계한다 | 권한 범위에 맞춘 현황 대시보드가 첫 화면에 표시된다 |
| 정상 | 파트별 통계의 '진행 중' 숫자를 클릭한다 | 해당 파트의 오늘 기준 진행 중 업무를 조회한다 | 진행 중 업무 목록 팝업이 열린다 |
| 예외 | 관리자가 로그인한다 | 전체 조직 범위로 현황을 집계한다 | 전 파트·전 조직 현황이 첫 화면에 표시된다 |
| 예외 | 미니 캘린더의 특정 날짜에 마우스를 올린다 | 그날 일정을 조회한다 | 해당 날짜 일정 툴팁이 뜨고, 클릭 시 상세 팝업이 열린다 |

## 4. 예시 (최소 2개, 실제 문장 인용)
- **입력**: "로그인 후 첫 화면에서 전체 업무 현황을 확인할 수 있어야 합니다."
- **기대 출력**: 오늘 일정·이번 주 일정·마감 임박 업무·마감 지연 업무·담당자별 진행 업무·파트별 진행률·미확인 업무지시·만료 예정 항목을 권한 범위에 맞춰 모은 첫 화면.
- **입력**: "통계 숫자 클릭 → 해당 상태의 업무 목록 팝업"
- **기대 출력**: 파트별 오늘 업무 통계(전체/시작 전/진행 중/완료)에서 숫자를 클릭하면 그 상태의 업무 목록이 팝업으로 뜬다.

## 5. 상세 요구 (번호 목록, "~해야 한다")
1. 로그인 후 첫 화면에서 오늘 일정·이번 주 일정·마감 임박 업무·마감 지연 업무·담당자별 진행 업무·파트별 업무 진행률·미확인 업무지시·최근 회의록·최근 공지 및 메모·휴가 및 부재자 현황·계약 및 비자 만료 예정·소프트웨어 라이선스 만료 예정·주요 알림을 표시해야 한다.
2. 관리자는 전체 조직을, 일반사용자는 본인이 권한을 가진 파트와 본인 업무만 볼 수 있도록 표시 범위를 권한에 따라 제한해야 한다.
3. 파트별 오늘 업무 통계를 전체 / 시작 전 / 진행 중 / 완료로 표시하고, 전체와 완료는 오늘 기준으로 집계해야 한다.
4. 통계 숫자를 클릭하면 해당 상태의 업무 목록 팝업을 띄워야 한다.
5. 파트별 미니 캘린더는 날짜 마우스 오버 시 그날 일정 툴팁을, 클릭 시 상세 팝업을 제공해야 한다.
6. '오늘 업무 현황' 섹션은 접기/펼치기가 가능해야 한다.

## 6. 다루는 데이터 (해당 시)
- 입력: 로그인 사용자의 권한·소속 파트 정보.
- 집계/조회: 업무(상태·마감일·진행률·담당자), 일정, 회의록, 공지·메모, 휴가·부재, 계약·비자·라이선스 만료일.
- 출력: 파트별 오늘 업무 통계 수치, 상태별 업무 목록, 일정 툴팁·상세.

## 7. 수용 기준 (체크박스 최소 3개, 참/거짓 판정 가능)
- [ ] 일반사용자로 로그인하면 본인 권한 파트·본인 업무만 대시보드에 표시되고 타 파트 항목은 보이지 않는다.
- [ ] 관리자로 로그인하면 전체 조직 현황이 대시보드에 표시된다.
- [ ] 파트별 오늘 업무 통계에 전체/시작 전/진행 중/완료 4개 숫자가 표시되고, 전체·완료가 오늘 기준으로 집계된다.
- [ ] 통계 숫자를 클릭하면 해당 상태의 업무 목록 팝업이 열린다.
- [ ] 미니 캘린더 날짜에 마우스 오버 시 일정 툴팁이, 클릭 시 상세 팝업이 뜬다.
- [ ] '오늘 업무 현황' 섹션을 접고 펼칠 수 있다.
