# 서식함 자료실

> 한 줄 정의: 직원이 회사 공용 양식·계약서·매뉴얼 파일을 폴더/카테고리로 올리고 검색·다운로드하면, 권한 범위 안에서 최신 버전 파일과 다운로드 이력을 얻는다.

## 1. 요구 배경
- **현재 불편(as-is)**: 인사·총무·회계·법무 서식과 계약서 양식·품의서·매뉴얼 등 각종 파일이 통합 관리되는 자리가 없어 어디에 최신본이 있는지 흩어져 있다. 요청서는 "회사에서 사용하는 각종 양식과 파일을 관리하는 자료실이 필요합니다"라고 명시한다. 파일이 여러 버전으로 돌아다니면 어느 것이 최신인지 판별하기 어렵다.
- **필요 이유**: 모든 공용 양식·자료를 분류·버전과 함께 한 곳에 모아 최신본을 권한별로 내려받게 하여 파일 산재와 버전 혼선을 없앤다.

## 2. 기능 정의
- **핵심 동작**: 파일을 폴더·카테고리로 분류해 업로드하고, 다운로드하게 한다.
- 파일명으로 자료를 검색하고, 작성자·등록일을 목록에 표시한다.
- 같은 자료의 파일을 교체하면 이전 버전을 보존해 버전 이력을 관리한다.
- 이전 버전 파일을 지정해 다운로드하게 한다.
- 권한 등급에 따라 파일 다운로드를 제한하고, 다운로드 이력을 기록한다.

## 3. 유저 플로우 (반드시 표, 케이스별)

| 케이스 | 행동 (사용자가 한다) | 과정 (시스템이 처리) | 결과 (산출물) |
|---|---|---|---|
| 정상 | 총무 담당자가 '총무 서식' 카테고리에 새 품의서 양식을 업로드한다 | 파일을 해당 카테고리에 저장하고 작성자·등록일을 기록한다 | 자료실 목록에 최신 양식이 등록됨 |
| 정상(교체) | 담당자가 기존 계약서 양식 파일을 새 파일로 교체한다 | 새 파일을 최신 버전으로 올리고 이전 파일을 이전 버전으로 보존한다 | 최신본 + 버전 이력이 남음 |
| 예외(권한) | 파일 다운로드 권한이 없는 사용자가 다운로드를 시도한다 | 권한 등급을 확인해 다운로드를 차단한다 | 다운로드 거부, 이력에는 시도 기록 없음(권한 밖 자료 미노출) |

## 4. 예시 (최소 2개, 실제 문장 인용)
- **입력**: "파일명 검색" — 사용자가 자료실에서 '인수인계서'라는 파일명으로 검색한다.
- **기대 출력**: 파일명에 '인수인계서'가 포함된 자료 목록을 작성자·등록일과 함께 반환한다.
- **입력**: "이전 버전 다운로드" — 사용자가 교체된 계약서 양식의 직전 버전을 요청한다.
- **기대 출력**: 보존된 이전 버전 파일을 내려주고 해당 다운로드를 이력에 남긴다.

## 5. 상세 요구 (번호 목록, "~해야 한다")
1. 인사·총무·회계·법무 서식, 계약서 양식, 품의서, 지출결의서, 인수인계서, 업무 매뉴얼, 회사소개서, 기타 자료 등으로 분류를 구성할 수 있어야 한다.
2. 파일을 폴더 및 카테고리로 관리하며 업로드·다운로드할 수 있어야 한다.
3. 파일명으로 자료를 검색할 수 있어야 한다.
4. 각 자료에 작성자와 등록일을 표시해야 한다.
5. 파일 교체 시 버전관리가 되어 이전 버전을 다운로드할 수 있어야 한다.
6. 권한 등급별로 파일 다운로드를 제한할 수 있어야 한다.
7. 다운로드 이력을 확인할 수 있어야 한다.

## 6. 다루는 데이터 (해당 시)
- 입력: 업로드 파일(양식·계약서·매뉴얼 등), 카테고리/폴더 분류값.
- 저장: 파일 본체와 버전 이력, 작성자·등록일 메타데이터, 다운로드 이력.
- 출력: 다운로드 파일(최신/이전 버전), 파일명 검색 결과 목록.

## 7. 수용 기준 (체크박스 최소 3개, 참/거짓 판정 가능)
- [ ] 파일을 특정 카테고리/폴더에 업로드하면 목록에 작성자와 등록일이 함께 표시된다.
- [ ] 파일명 일부로 검색하면 해당 문자열을 포함한 자료만 결과에 나온다.
- [ ] 같은 자료의 파일을 교체한 뒤 이전 버전을 지정해 다운로드할 수 있다.
- [ ] 파일 다운로드 권한이 없는 사용자는 해당 파일을 다운로드할 수 없다.
- [ ] 파일을 다운로드하면 누가 언제 받았는지 다운로드 이력에 기록된다.
