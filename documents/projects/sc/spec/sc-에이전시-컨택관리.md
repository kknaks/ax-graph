# 에이전시 컨택관리

> 담당자가 인플루언서·에이전시·병원 등 외부 파트너의 컨택 정보를 등록·추적하면, 컨택 예정부터 계약 완료까지의 진행 현황과 전환율을 한 화면에서 관리한다.

## 1. 요구 배경
- **현재 불편(as-is)**: 인플루언서·크리에이터·에이전시·병원 등 외부 파트너와의 컨택이 담당자별로 흩어져 관리되어 누구를 언제 어디까지 컨택했는지 통합 추적이 어렵다. 요청서는 "인플루언서, 크리에이터, 에이전시, 병원 및 외부 파트너의 컨택 현황을 관리하는 CRM 형태의 기능이 필요합니다"라고 명시한다.
- **필요 이유**: 컨택 항목·상태·다음 연락일·대화 내역을 한 곳에 모아 관리하면 컨택 누락과 중복을 막고 계약 전환까지의 흐름을 추적할 수 있다.

## 2. 기능 정의
- 컨택일·매체·계정 링크·계정 아이디·이름/활동명·국적·언어·팔로워 수·평균 조회수·담당자·컨택 경로·제안 내용·희망조건·상태·다음 연락 예정일·메모·첨부파일을 관리 항목으로 등록한다.
- 컨택 예정 → 메시지 발송 → 답변 대기 → 응답 → 소통 중 → 미팅 조율 → 제안서 발송 → 계약 대기 → 계약 완료(및 보류·거절)의 상태로 컨택 진행을 흐름 관리한다.
- 매체별·국가별·담당자별·상태별·팔로워 구간별로 컨택 목록을 필터링한다.
- 다음 연락 예정일 기준으로 담당자에게 알림을 표시하고, 중복 계정을 확인한다.
- 파트너별 대화 내역(보낸 메시지·받은 메시지 구분)을 기록하고 컨택 전환율 통계를 산출한다.

## 3. 유저 플로우 (반드시 표, 케이스별)

| 케이스 | 행동 (사용자가 한다) | 과정 (시스템이 처리) | 결과 (산출물) |
|---|---|---|---|
| 정상 | 신규 인플루언서 컨택을 등록하고 메시지 발송 후 상태를 갱신한다 | 관리 항목을 저장하고 상태를 '메시지 발송'으로 반영, 다음 연락 예정일을 기록 | 컨택 카드가 목록에 생성되고 진행 상태·다음 연락일이 표시됨 |
| 예외 | 이미 등록된 계정 링크로 컨택을 등록하려 한다 | 계정 링크/아이디 중복을 확인해 기존 컨택과 겹침을 경고 | 중복 계정 알림이 표시되어 중복 등록을 방지 |

## 4. 예시 (최소 2개, 실제 문장 인용)
- **입력**: 요청서의 관리 항목 "컨택일 / 매체 / 계정 링크 / 계정 아이디 / 이름 또는 활동명 / 국적 / 언어 / 팔로워 수"를 담은 신규 파트너 정보.
  - **기대 출력**: 해당 항목이 채워진 컨택 카드와 초기 상태 '컨택 예정'.
- **입력**: "다음 연락 예정일"이 오늘로 지정된 컨택 건.
  - **기대 출력**: 담당자에게 다음 연락일 알림이 표시되고 해당 컨택이 후속 대상으로 노출.

## 5. 상세 요구 (번호 목록, "~해야 한다")
1. 컨택일·매체·계정 링크·계정 아이디·이름/활동명·국적·언어·팔로워 수·평균 조회수·담당자·컨택 경로·제안 내용·희망조건·상태·다음 연락 예정일·메모·첨부파일을 등록·수정할 수 있어야 한다.
2. 컨택 상태를 컨택 예정·메시지 발송·답변 대기·응답·소통 중·미팅 조율·제안서 발송·계약 대기·계약 완료·보류·거절로 선택·전환할 수 있어야 한다.
3. 매체별·국가별·담당자별·상태별·팔로워 구간별로 컨택 목록을 필터링할 수 있어야 한다.
4. 다음 연락 예정일 기준으로 담당자에게 알림을 표시해야 한다.
5. 계정 링크·아이디 기준으로 중복 계정을 확인하고 계정 링크 바로가기를 제공해야 한다.
6. 파트너별 대화 내역을 보낸 메시지와 받은 메시지로 구분하여 기록할 수 있어야 한다.
7. 컨택 목록을 엑셀로 다운로드하고 컨택 전환율 통계를 제공해야 한다.

## 6. 다루는 데이터 (해당 시)
- 입력/저장: 파트너 컨택 항목(계정 링크·아이디·팔로워 수 등), 컨택 상태 이력, 다음 연락 예정일, 대화 내역, 첨부파일.
- 출력: 필터링된 컨택 목록, 엑셀 다운로드, 컨택 전환율 통계.

## 7. 수용 기준 (체크박스 최소 3개, 참/거짓 판정 가능)
- [ ] 컨택 항목을 등록하면 목록에 컨택 카드가 생성되고 상태·다음 연락일이 표시된다.
- [ ] 매체·국가·담당자·상태·팔로워 구간 필터로 목록을 좁힐 수 있다.
- [ ] 동일 계정 링크/아이디 등록 시 중복 경고가 표시된다.
- [ ] 다음 연락 예정일이 도래하면 담당자에게 알림이 표시된다.
- [ ] 컨택 목록을 엑셀로 다운로드하고 컨택 전환율 통계를 조회할 수 있다.
