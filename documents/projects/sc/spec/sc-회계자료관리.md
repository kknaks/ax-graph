# 회계자료관리

> 회계 담당자가 회계업무 시트·파일·업무절차·인수인계 자료를 한 메뉴에 등록·관리하면, 흩어진 회계 자료와 업무 흐름을 한곳에서 찾고 시각화된 절차로 확인한다.

## 1. 요구 배경
- **현재 불편(as-is)**: 월결산·세금계산서·부가세·급여 등 회계업무에 쓰는 구글시트·드라이브 파일과 업무절차·인수인계 자료가 개인·부서별로 흩어져 있어 담당자·마감일·업무 순서를 한눈에 파악하기 어렵다. 요청서는 "회계업무에 사용하는 시트, 파일, 업무절차 및 인수인계자료를 관리하는 메뉴가 필요합니다"라고 명시한다.
- **필요 이유**: 회계 자료·링크·절차·인수인계 정보를 분류별로 한 메뉴에 모으고 업무흐름도로 절차를 시각화해 담당자 교체·마감 관리 시 누락을 없앤다.

## 2. 기능 정의
- 회계 자료를 분류(월 결산·매출/매입·세금계산서·법인카드·원천세·부가세·급여·자금일보·자회사 결산·외부감사·연결회계·세무조정·회계 업무 매뉴얼)별로 등록한다.
- 외부 구글시트·드라이브 링크를 등록하고, 별도 파일을 업로드한다.
- 각 자료에 담당자·업무 주기·마감일·업무 순서·체크리스트·주의사항·관련 자료·인수인계 담당자를 함께 기록한다.
- 자료의 수정이력을 남겨 누가 언제 무엇을 바꿨는지 확인한다.
- 회계 업무 절차를 코드 기반 순서도(플로우 다이어그램)로 시각화하고 주제별 흐름 기록을 관리한다.

## 3. 유저 플로우 (반드시 표, 케이스별)

| 케이스 | 행동 (사용자가 한다) | 과정 (시스템이 처리) | 결과 (산출물) |
|---|---|---|---|
| 정상 | 회계 담당자가 '부가세' 분류에 구글시트 링크를 등록하고 담당자·업무 주기·마감일·체크리스트를 입력 | 링크·메타정보를 저장하고 수정이력에 기록, 마감일을 등록 | 분류별 목록에 노출되는 회계 자료 항목 1건과 수정이력 |
| 정상 | 담당자가 '월 결산' 절차를 코드 기반 순서도로 작성 | 코드를 순서도(플로우 다이어그램)로 렌더링해 저장 | 주제별로 조회 가능한 업무 흐름도 |
| 예외 | 인수인계 담당자를 지정하지 않고 저장 시도 | 인수인계 담당자 미지정 상태로 저장하되 인수인계 정보 누락을 표시 | 인수인계 담당자 미지정 표시가 붙은 자료 항목 |

## 4. 예시 (최소 2개, 실제 문장 인용)
- **입력**: "외부 구글시트 또는 드라이브 링크 등록" — 담당자가 부가세 신고용 구글시트 링크를 등록한다.
- **기대 출력**: 부가세 분류 아래에 해당 구글시트 바로가기와 담당자·마감일·체크리스트가 묶여 표시된다.
- **입력**: "업무 흐름도(플로우 다이어그램): 코드 기반 순서도로 업무 절차를 시각화" — 담당자가 월 결산 절차를 코드로 작성한다.
- **기대 출력**: 작성한 코드가 순서도로 렌더링되어 월 결산 주제의 업무 흐름도로 저장·조회된다.

## 5. 상세 요구 (번호 목록, "~해야 한다")
1. 회계 자료를 분류(월 결산·매출/매입·세금계산서·법인카드·원천세·부가세·급여·자금일보·자회사 결산·외부감사·연결회계·세무조정·회계 업무 매뉴얼) 기준으로 등록·조회할 수 있어야 한다.
2. 외부 구글시트·드라이브 링크 등록과 파일 업로드를 모두 지원해야 한다.
3. 각 자료에 담당자·업무 주기·마감일·업무 순서·체크리스트·관련 자료·주의사항을 함께 기록할 수 있어야 한다.
4. 인수인계 담당자를 지정할 수 있어야 한다.
5. 자료의 수정이력이 남아 누가 언제 수정했는지 확인할 수 있어야 한다.
6. 회계 업무 절차를 코드 기반 순서도로 시각화하고 주제별 흐름 기록을 관리할 수 있어야 한다.

## 6. 다루는 데이터 (해당 시)
- 입력: 회계 자료 분류, 외부 구글시트·드라이브 링크, 업로드 파일, 담당자·업무 주기·마감일·업무 순서·체크리스트·주의사항·인수인계 담당자, 업무 흐름도 코드.
- 저장: 자료 메타정보, 첨부 파일·링크, 수정이력, 주제별 업무 흐름도.
- 출력: 분류별 회계 자료 목록, 렌더링된 순서도.

## 7. 수용 기준 (체크박스 최소 3개, 참/거짓 판정 가능)
- [ ] 분류별로 회계 자료를 등록하고 해당 분류에서 조회할 수 있다.
- [ ] 외부 구글시트·드라이브 링크와 업로드 파일을 함께 등록할 수 있다.
- [ ] 자료에 담당자·업무 주기·마감일·체크리스트·인수인계 담당자를 입력하고 저장할 수 있다.
- [ ] 자료 수정 시 수정자·수정일시가 수정이력에 남는다.
- [ ] 코드 기반 순서도로 작성한 업무 절차가 순서도로 렌더링되어 주제별로 조회된다.
