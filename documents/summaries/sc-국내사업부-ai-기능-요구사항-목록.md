## 1. 병원 매출 동향 자동 분석

- **대상 사용자**: 마케터
- **입력**: 매일/매월 입력하는 병원 매출 데이터
- **원하는 출력**: 전체적인 시장 흐름을 자동 요약
- 세부 니즈:
  - 어느 지역 매출이 떨어지는지
  - 어떤 진료 과목이 잘 나가는지
  - 한눈에 보이는 대시보드 형태

## 2. 메모 & 회의록 자동 찾아주기

- **대상**: 자유 형식으로 쌓인 메모·회의록
- **원하는 동작**: 자연어 질의로 원하는 내용을 정확하게 검색
- 예시 질의:
  - "7월 7일 SC 임원회의 내용 찾아줘"
  - "지난주 A고객사 만난 내용 보여줘"
- 조건: 혼합된 기록에서도 헷갈리지 않고 정확한 답변 반환

## 3. 직원 게릴라 미션 및 일정 관리

- **목적**: 갑작스러운 지시사항·미션이 묻히지 않도록 관리
- **입력**: 대략적인 지시 내용 + 날짜
- **원하는 동작**:
  - 해당 직원 일정에 자동 등록
  - 중간중간 잊지 않게 알림 발송
- 예시: "A담당자한테 경쟁사 조사하라고 해줘"

## 4. 대화형 제안서 작성 도우미

- **입력**: 회의록 또는 아이디어
- **원하는 동작**:
  1. AI가 부족한 부분을 사용자에게 인터뷰(질문) 방식으로 보완
  2. 기존에 잘 된 제안서 양식을 참고
  3. 새 제안서 파일로 완성
- **특이사항**: 단순 생성이 아닌 대화형(인터뷰) 방식 강조

## 5. 부서 공유 캘린더 및 업무일지 (노션 대용)

- **목적**: 부서원 일과·프로젝트 진행 상황을 한곳에서 관리
- 포함 기능:
  - 개인별 하루 업무일지 작성
  - 부서 전체 일정 공유 달력
  - 직원별/프로젝트별 진행 상황판
- **포지셔닝**: 노션 대체 용도로 명시

## 6. 신입사원용 피부/성형 시술 지식인

- **대상 사용자**: 신입사원
- **콘텐츠**: 피부과·성형외과 주요 시술의 특징, 주의사항 등
- **원하는 인터페이스**: 자연어 질문에 친절하게 답하는 검색/대화창
- **운영 방식**: 시술 데이터를 사전에 입력해두고 Q&A 형태로 제공

## 7. 매일 보는 의료 뉴스 모음

- **원하는 동작**: 의료 관련 최신 기사를 매일 자동 수집
- **출력 형태**: 핵심만 짧게 요약한 매거진 스타일 리포트
- **주기**: 매일

## 8. 프로그램 내 문자 발송

- **목적**: 영업·업무 연락용 문자를 프로그램 안에서 직접 발송
- 세부 요건:
  - 자주 쓰는 문자 양식 저장 기능
  - 외부 문자 발송 서비스 연동

## 9. 마케팅 순위 자동 확인 (루틴 업무 자동화)

- **현재 방식**: 매일 사람이 직접 검색하여 확인
- **자동화 대상 항목**:
  - 네이버 지도(플레이스) 순위
  - 블로그 상위 노출 순위
  - 체험단 글 순위
- **출력**: 매일 자동 체크 후 리포트 형태로 제공

## 10. 체험단 블로그 원고 자동 검수

- **목적**: 블로거 작성 글의 가이드라인 준수 및 법적 리스크 사전 차단
- 검수 항목:
  - 제공된 가이드라인대로 작성했는지 체크
  - **의료법**에 걸릴 위험 단어·문구 자동 감지 및 표시(빨간 줄)
  - 사진 삽입 여부 확인
