## 문서 개요

- 작성 주체: **SC 디자인팀·영상팀** (전지연 이사)
- 성격: 내부 업무 관리 도구에 대한 기능 요구사항 목록

---

## 1. 회의록 자동 녹음 및 요약 기능

- 회의를 자동으로 녹음하고 요약하는 기능
- **이전 회의록 불러오기**: 관련 업무 진행 시 과거 회의록을 참조할 수 있는 기능
- **관련자 태그 및 공유**: 관련자 태그 시 해당 회의록을 공유하되, 필요한 부분만 선택적으로 노출할 수 있는 권한 제어 기능
- **관련 업무 진행 체크**: 회의에서 도출된 업무 항목의 진행 여부를 체크하는 기능

---

## 2. 자리 비움 상태 확인 기능

- 대상 상태 유형: 반차, 연차, 휴가, 미팅, 잠시 자리 비움 등
- **구성원 현재 상태 확인**: 작업 진행 중 관련자가 자리에 있는지 없는지 실시간으로 확인할 수 있는 기능

---

## 3. 연관 업무 연결 기능

- **수정 요청 시 이전 작업 연결**
  - 요청자, 작업자, 작업 경로를 함께 확인할 수 있어야 함
- **신규 요청 시 유관 작업 연결**
  - 신규 요청이 들어올 때 기존의 관련 작업과 자동 또는 수동으로 연결
  - 요청자, 작업자, 작업 경로 확인 필요

---

## 4. 업무 진행·분배 확인 및 알람 기능

- **담당 디자이너 확인**: 기획안 요청 후 본인 작업을 어느 디자이너가 담당하는지 확인하는 기능
- **작업 상태 확인**: 작업이 이미 진행 중인지, 예정 상태인지 구분하여 확인할 수 있는 기능 (기획안 수정 시 특히 유용)
- **완료일 변경 알람**: 완료일이 변경될 경우 작업자와 요청자에게 동시에 알람 발송
- **완료일 D-1 기본 알람**: 완료일 하루 전부터 기본 알람이 설정되며, 작업자와 요청자 모두에게 동시 발송 — 업무 누락 방지 목적
