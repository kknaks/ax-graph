## 1. 프로젝트 부트스트랩 (CLI)

- Getting Started 가이드에서 npm 명령어를 복사해 실행하면 **VoltAgent 앱이 즉시 부트스트랩**됨
- 설정 중 AI 프로바이더를 선택 → 데모에서는 **OpenAI** 선택
- 프로젝트 열기 후 `.env` 파일에 **OpenAI API 키** 추가 필요 (데모에서는 OpenAI 키만으로 충분)
- `npm run` 명령으로 서버 시작 → 서버가 올라오면 **VoltOps 플랫폼에 자동 연결**됨

## 2. 기본 에이전트 — 날씨 에이전트

### 동작 흐름
1. 부트스트랩 프로젝트에는 **날씨 질문에 답하는 간단한 에이전트**가 포함됨
2. 테스트 쿼리 예시: "What is the weather in San Francisco today?"
3. 에이전트가 입력을 받아 **weather tool을 호출**하고 답변을 생성

### 도구(Tool)의 역할
- 도구는 에이전트를 유용하게 만드는 핵심 요소
- 단순 텍스트 생성이 아니라 **실제 액션**을 수행 가능
  - 날씨 확인, 데이터베이스 검색, 이메일 발송 등

### VoltOps 콘솔에서 확인 가능한 항목
- 각 에이전트의 **상태, 입력, 출력, 시스템 프롬프트**
- Input 섹션: 사용자가 입력한 정확한 질문
- Messages 섹션: LLM에 전송된 내용
- Output 섹션: 사용자 통계가 포함된 전체 응답

### 에이전트 코드 구조
- 에이전트 정의 시 **instructions(지시사항)** 포함
- 모델: `OpenAI GPT-4o mini` (다른 모델로 교체 가능)
- 에이전트 등록 → 워크플로우 추가 → **VoltOps 플랫폼 연결** 순서로 구성

## 3. 워크플로우 — 비용 승인 워크플로우 (Expense Approval)

### 워크플로우 개념
- 워크플로우 = **단계들의 체인(chain of steps)**, 각 단계가 결과를 다음 단계로 전달
- `createWorkflowChain` 메서드로 체인을 구성

### 비용 승인 워크플로우 로직

| 금액 조건 | 동작 |
|---|---|
| $500 미만 | 자동 승인 → 워크플로우 계속 진행 |
| $500 초과 | **일시정지(suspend)** → 매니저 승인 대기 |

### 데모 시나리오 1 — $250 (한도 이하)
1. $250 입력 후 워크플로우 실행
2. 한도 이하이므로 **자동 승인**
3. Check Approval 단계가 승인됨으로 표시 → Process Decision 단계 → 워크플로우 완료

### 데모 시나리오 2 — $750 (한도 초과)
1. $750 입력 후 워크플로우 실행
2. **워크플로우 일시정지(suspend)** — Check Approval 단계에서 대기
3. 매니저 응답 없이는 진행 불가
4. 매니저 결정으로 **재개(resume)** → Process Decision → 워크플로우 완료
5. 타임라인에서 각 이벤트를 단계별로 확인 가능 → **효율적인 디버깅 지원**

### 워크플로우 코드 구조
- **Step 1**: 승인 필요 여부 확인
  - $500 미만 → 자동 승인
  - $500 초과 → `suspend` 호출로 워크플로우 일시정지, 비용 상세 정보와 함께 매니저 승인 대기
- **Step 2**: 결정 확정(finalize) 및 결과 로깅

## 4. 프로덕션 모드 및 옵저버빌리티

- **Enable 버튼** 클릭으로 프로덕션 모드 활성화
- 프로덕션 사용자가 트리거한 정확한 플로우를 단계별로 실시간 확인 가능
- 대시보드에서 **옵저버빌리티 메트릭** 실시간 추적
  - 에이전트 전반적인 상태(health) 및 효율성 개요 제공
- **트레이스(traces)** 추적으로 프로덕션 이슈 디버깅 가능
  - 사용자가 얻은 결과와 그 이유를 정확히 파악
- 과거 실행(past runs) 조회: 입력·출력·**비용(costs)** 확인
  - 특정 실행 클릭 시 단계별 상세 흐름 확인 가능

## 5. 마무리 및 확장 가능성

- 에이전트와 워크플로우가 함께 작동하여 **원활한 데이터 플로우 + 내장 옵저버빌리티** 제공
- 향후 확장 방향:
  - 더 많은 에이전트 추가
  - 조건부 로직 추가
  - Human-in-the-Loop 승인 단계 추가
