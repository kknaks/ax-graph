## 1. 개발 목적

- 경영관리부·관련 TF·자회사 업무를 단일 시스템에서 통합 관리하는 **사내 통합 업무관리 웹서비스** 개발
- 단순 게시판·일정관리가 아닌 실무 전 영역 통합
  - 부서·파트별 업무관리 / 업무지시·진행상황
  - 회의록·후속 업무 생성
  - 인사정보·직원 문서 / 근태·야근수당
  - 총무 자산·소프트웨어 관리
  - 계약서·법무자료 / 회계자료·업무 매뉴얼
  - TF 프로젝트·외부 파트너 컨택
  - 자회사별 일정·업무·자료
- **플랫폼**: PC 웹 우선, 모바일 조회·등록·수정 가능한 반응형 웹

## 2. 사용자 및 권한관리

### 2-1. 로그인
- 직원별 개인 아이디/비밀번호
- 최초 로그인 시 비밀번호 직접 설정, 변경·초기화

- 로그인 및 주요 정보 수정 이력 저장

### 2-2. 사용자 권한 — 조직 구조
- **경영관리부**: 회계·인사·총무·법무·비서 파트
- **TF 프로젝트**: 라이브 에이전시, 환자유치 에이전시, 기타 신규 TF 추가 가능
- **자회사**: 회사별 독립 업무 공간, 관리자가 자유롭게 추가·수정·삭제

### 2-3. 권한 등급

- 사용자별 세부 권한 설정: 메뉴 조회, 자료 등록·수정·삭제, 엑셀·파일 다운로드, 직원 개인정보·급여정보 조회, 관리자 설정 접근

## 10. 근태 및 야근수당 계산

- 에스원(S-1) 출입기록 엑셀 업로드 → 직원별 근태·야근시간 자동 계산
- 계산 항목: 출퇴근 시각·총 근무시간·기본·연장·휴일·야간 근무시간·지각·조퇴·식사시간 공제·직원별 시간당 수당·야근수당·월별 누적·전월 이월
- 예외 설정: 직원별 출근 인정시간, 유연·시간제·시프트 근로자, 외근·출장·휴가·반차·지각 제외 사유·특정 날짜 예외
- 계산 결과 수정 시 수정자·수정 전후 값 기록, 엑셀 다운로드

### ▣ 현행 시스템 — 야근수당 계산 상세 ⚠️ (임의 변경 금지)

**① 입력 파일**
- 에스원 출입기록 엑셀 — 이벤트가 "입실"만 기록됨
- 해당일 **첫 태그 = 출근**, **마지막 태그 = 퇴근**으로 판정

**② 정산 기간 / 지급일**
- 정산기간: **전월 25일 ~ 당월 24일**, 지급일: **당월 말일**
- 태그 날짜의 '일'이 25일 이상이면 **다음 달 귀속**

**③ 출근 시각별 정규 근무 종료**

| 출근(첫 태그) 시각 | 정규 근무 종료 |
|---|---|
| ~ 08:59 | 18:00 |
| 09:00 ~ 09:29 | 18:30 |
| 09:30 ~ 09:59 | 19:00 |
| 10:00 이후 | 지각 — 야근 미인정 |

- 지각 예외자(설정 지정): 10:00 이후 출근이어도 19:00 초과분은 야근 인정

**④ 야근시간 계산**
- 퇴근 − 정규 근무 종료, **1분 단위(초 버림)**
- 정규 종료 이전 퇴근 시 0

**⑤ 반차**
- 수동 표시, 출근을 정규 구간으로 스냅 후 실제 시작 + 4시간 = 근무 종료, 이후가 야근
- 오후반차 시작: 오전 브래킷(09:00/09:30/10:00), 오전반차 시작: 오후 브래킷(14:00/14:30/15:00)
- 누락일은 수기 입력

**⑥ 수당·이월**
- 정수 시간당 **₩10,320**(최저시급)
- 월 합계에서 **정수 시간만 지급**, 1시간 미만 잔여분은 **다음 달 이월**
  - 예: 12시간 47분 → 12시간분 지급, 47분 이월

**⑦ 편집 기능**
- 정산기간 전체 날짜 표시(누락 평일 빨강 표시)
- 출퇴근·수동야근 직접 입력
- 지급시간·수당·이월 수동 덮어쓰기(빈칸 = 자동)
- 자동저장/복원, 월별 스냅샷 저장·재조회

**⑧ 검증 기준값**
> 2026년 6월(5/25~6/24) 15명 = 총 **₩3,384,960** — 개발 후 이 수치로 대조 테스트 권장

## 11. 점심 식대 계산

- CSV·엑셀 업로드 → 직원별 식대·월별 총액 계산
- 기능: 직원명 자동 매칭, 신청일별 내역, 직원·부서별 식사 횟수, 단가 설정, 개인·회사 부담 구분, 중복·미등록 직원 확인, 월별 합계, 엑셀 다운로드, 재업로드

### ▣ 현행 시스템 — 식대 계산 상세 ⚠️ (임의 변경 금지)

**① 정산 흐름**
1. **더에쓰씨(THE SC)**가 급식업체에 전액 결제
2. 각 회사에 총 식대(청구액) 청구
3. 각 회사가 직원 급여에서 직원 부담금 차감

**② 입력 CSV 컬럼**
- 날짜 / 이름 / 회사 / 부서 / 상태 / 신청일시
- **상태='paid'인 행만 집계**, 1행 = 1식

**③ 회사 분류(4개 그룹)**

| 조건 | 정산 그룹 |
|---|---|

**⑤ 계산식**
- 회사 청구액 = Σ(식수 × 해당일 식대총액)

- 회사 부담금 = 회사 청구액 − 직원 부담금
